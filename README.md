# segment_control

A segment control based on Tabbar implementation  
一个基于TabBar控件改装的Segment控件

## Demo   
![][image-1]

## Features
- [x] **可以自定义验证码长度**  
- [x] **当全部输入完成后,会自动调用协议方法,通过代理返回验证码及上下文context**  
- [x] **可以通过verCode属性获取输入的验证码**
- [x] **用户自定义控件的尺寸**

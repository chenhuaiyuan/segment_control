# segment_control

A segment control based on Tabbar implementation  
一个基于TabBar控件改装的Segment控件

## Demo   
![][image-1]

## Features
- [x] **自定义选项数据**  
- [x] **自定义样式,包括选中和未选中的背景色,字体样式以及边框颜色,圆角等**  
- [x] **回调返回选择的选项索引以及标题**
- [x] **可以设置未选中样式**
